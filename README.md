# Front Libs

A convenient aggregator for the front end libraries used in VTEX projects.

Available at http://io.vtex.com.br/front-libs/

Libraries included:

- jQuery 1.8.3
- Bootstrap 2.3.2
- Font Awesome 3.2.1
- Front i18n 0.3.2
- Front Utils 1.0.1
- VTEX Events 1.0.0
- Underscore 1.5.2
- Dustjs-linkedin 2.2.2
- Dustjs-linkedin-helpers 1.1.1
- Curl 0.8.4
